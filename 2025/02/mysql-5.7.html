Installing MySQL 5.7 Without Apt

Feb 25, 2025

Steps to Install MySQL 5.7 Manually

  1. Download MySQL 5.7 Tarball
    Go to the MySQL Community Downloads page and download the correct tarball for your system.

  2. Extract the Tarball
    Use the following command to extract the downloaded tarball: ```bash tar -xvzf mysql-5.7.x-linux-glibc2.12-x86_64.tar.gz

  3. Move to desired directory Move the extracted MySQL folder to your desired directory (for example, /usr/local/mysql): ```bash sudo mv mysql-5.7.x-linux-glibc2.12-x86_64 /usr/local/mysql

  4. Create MySQL User and Group If the MySQL user and group don’t already exist, create them with the following commands: ```bash sudo groupadd mysql sudo useradd -r -g mysql -s /bin/false mysql

  5. Set Permissions for MySQL Directory Set the correct ownership of the MySQL directory: ```bash sudo chown -R mysql:mysql /usr/local/mysql

  6. Initialize the MySQL Data Directory Go into the MySQL directory and initialize the data directory: ```bash cd /usr/local/mysql sudo ./bin/mysqld –initialize –user=mysql –datadir=/usr/local/mysql/data

  7. Remove Old Symlink (If Applicable) If you have a newer version of MySQL installed (e.g., MySQL 8.0), remove the old symlink and create a new one: ```bash sudo rm /usr/bin/mysql sudo ln -s /usr/local/mysql/bin/mysql /usr/bin/mysql

  8. Verify ```bash mysql –version
